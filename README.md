[AutoFixture.AutoFoq](https://www.nuget.org/packages/AutoFixture.AutoFoq) is an extension for AutoFixture that turns your fixture into an auto-mocking container using Foq. It automates the creation of mocks for abstract classes and interfaces, letting you focus on your test's intent instead of tedious setup.

> [!WARNING]
> This package is intended only for legacy projects that are still using Foq and is no longer actively maintained
> Use at your own risk.
